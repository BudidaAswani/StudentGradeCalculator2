# 📊 Student Grade Management System (Java)

## 📌 Description
This is a console-based Java application that allows users to input student details and marks, calculate the average, and assign grades based on performance.

---

## 🚀 Features
- Input student name and marks for 3 subjects
- Calculate average marks
- Assign grade (A / B / C / F)
- Display structured student report
- Handles accurate calculations using double

---

## 🛠️ Technologies Used
- Java
- OOP Concepts (Methods)
- Conditional Statements

---

## ▶️ How to Run
1. Open the project in Eclipse or any Java IDE
2. Compile and run the `StudentGradeCalculator.java` file
3. Enter student details and marks
4. View the generated report
